How it really works

Pressup uses the WordPress REST API to hook in to the sites. It sends requests every couple of minutes to identify whether there are any new Core, Language, Theme or Plugin updates across any of your connected sites. It pulls all these updates in to a feed that sits in your Mac menu bar where you can easily update.
